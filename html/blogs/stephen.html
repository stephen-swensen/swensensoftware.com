x11 forwarding (linux guest, windows host)

I'm playing with sdl2 and rust. In my setup, I ssh from Bash for Windows (WSL) into Ubuntu Server running inside Virtual Box. In order to run my sdl2 app over my ssh connection, I needed to:

  • Install xorg in the guest: sudo apt install xorg
  • Install Xming on Windows
  • Set the DISPLAY variable in my WSL host session: export DISPLAY=localhost:0.0
  • Use the -X flag in my ssh command
